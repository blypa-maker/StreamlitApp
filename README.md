### How to run it on your own machine

Steps: 

1. Create conda env 

        conda create --name <env> --file requirements.txt

2. Create an ENV variable for openai api key

        set OPENAI_API_KEY=your_api_key

3. Run streamlit app 

        streamlit run streamlit_app.py

4. Run with docker 

        docker compose up -d
